[**Voir la démo**](https://dzeko003.github.io/projet-6-2-portfolio-personnel/)

- **Déploiement** : GitHub Pages

- Filtrage dynamique des projets
- Formulaire de contact
- Section blog & expérience
- Animations CSS et micro-interactions
